## Message 2 (par MakkuSiu#9166 [692721406219124806])

**En résumé, les positions en CSS :**

La position **_relative_** : L'élément ayant cette position **est** dans le **flux**. Les valeurs qu'on lui donne avec top left bottom ou right décale l'élément par rapport à lui même.

La position **_absolute_** : L'élément ayant cette position **n'est pas** dans le **flux**. Les valeurs que l'on lui donne avec top left bottom ou right positionne l'élément depuis son bord et le bord de son plus proche parent **positionné**(right : 10px; positionnera l'élément absolute à 10px d'écart entre le bord droit de son parent positionné et son bord droit).

La position **_fixed_** : L'élément ayant cette position **n'est pas** dans le **flux**. Les valeurs que l'on lui donne avec top left bottom ou right positionne l'élément depuis son bord et le bord du viewport (la zone d'affichage de votre page)(right : 10px; positionnera l'élément absolute à 10px d'écart entre le bord droit de sa page son bord droit. Il restera donc fixer a cette position sur la zone d'affichage même si un scroll est effectué).

La position **_sticky_** difficilement expliquable : allez vous informer ici : https://developer.mozilla.org/fr/docs/Web/CSS/position **en très gros** résumé c'est un mélange entre la position **relative** et la position **fixed**.

- la déclaration d'une variable sans let, const ou var
- l'affectation de valeur à une variable ou une propriété non accessible en écriture

## Message 4 (par Yann H.#8396 [523536894580228112])

Display inline, inline-block et block, sur plusieurs divs par exemple :

- `display: inline` -> va aligner horizontalement les divs à la suite d'autres éléments, mais impossible de jouer sur le height ou le width ou le margin de ces divs. (padding fonctionne)
- `display: inline-block` -> va aligner horizontalement les divs, et possibilité de jouer sur le height ou le width margin et padding de ces divs.
- `display: block` -> va forcément passer à la ligne chaque div, et possibilité de jouer sur le height ou le width margin et padding de ces divs.
